# Home
## 説明

| コマンド | 内容 |
| ---- | ---- |
| /hogo help | ヘルプを表示します。 |
| /money pay プレイヤー名 金額 | 指定した人に指定された金額を送金します。 |

## 保護の仕方
まず、保護をするには範囲を指定します。  
//wand と実行して木の斧を入手します。  
その斧を使用して保護したい範囲の始点を左クリックして次に終点を右クリックしてください。  

もしここで今指定した土地がいくらかかるのか確認したい場合は  
`/hogo price`と実行し確認することができます。
